# Quickstart Validation — Example Feature

## Setup
Open the repository and inspect:
- /specs/_templates
- /specs/000-example-feature

## Happy path
1. Copy `specs/000-example-feature` into a new folder.
2. Replace placeholder content with a real feature description.
3. Confirm the new folder contains spec, plan, tasks, and quickstart.

## Failure path
1. Omit the plan or tasks file.
2. Confirm the workflow now feels incomplete and cannot guide implementation.

## Edge case
1. Use this workflow for a tiny bugfix.
2. Confirm that a lighter version may be preferable for very small changes.
